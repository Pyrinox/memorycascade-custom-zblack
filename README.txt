== Description ==
ZBlack - A Responsive Blog Theme for WordPress. This is a responsive and retina-ready Wordpress theme. ZBlack is a theme installed and customized easily.

* Mobile-first, Responsive Layout
* Custom Colors
* Custom Header
* Social Links
* Menu Description
* Post Formats
* and more

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button -> Click the 'Upload Theme' button and upload zblack.zip.
2. Click on the 'Activate' button to use your new theme right away.
3. Go to documentation for a guide on how to customize this theme.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
